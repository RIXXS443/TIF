# Web Presentacion TIF

Micrositio interactivo para exponer el Trabajo Integrador Final sobre Familia Bercomat.

## Stack

- TypeScript
- HTML
- CSS
- Bootstrap
- Leaflet
- Python opcional para servir el build final

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

Este proyecto queda preparado para publicar en:

`https://rixxs443.github.io/TIF/`

El despliegue se ejecuta automaticamente con GitHub Actions al hacer push a `main`.

## Servir el build con Python

Cuando Python este disponible en el equipo:

```bash
python server.py
```
